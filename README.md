# Weather App

Provides real-time weather updates, including temperature, humidity, and air quality information. This feature is powered by the OpenWeather API, offering accurate and up-to-date environmental data.

## Features

- **Real-Time Weather Data**: Fetch current temperature, humidity, and air quality information.
- **Geolocation Support**: Automatically detect and fetch weather data for the user's location.
- **City Search**: Retrieve weather details for any city worldwide.
- **User-Friendly Interface**: A clean and responsive UI for seamless interaction.

## Technologies Used

- **Frontend**: React, JavaScript, Tailwind CSS
- **API**: OpenWeather API
- **Deployment**: Vercel

### Environment Variables

Create a `.env` file in the root directory and add the following:

```plaintext
REACT_APP_OPENWEATHER_API_KEY=your_openweather_api_key
```
Replace `your_openweather_api_key` with your actual API key from OpenWeather.
